Objetivo: 

Desenvolver um plano de testes completo para um sistema hipotético, partindo do levantamento de requisitos, passando pela elaboração dos casos de teste, execução dos testes e documentação dos resultados. O trabalho deve incluir a aplicação dos seguintes tipos de testes: 

Teste de Unidade 
Teste de Integração 
Teste de Sistema 
Teste de Aceitação 
Teste de Regressão 
Teste de Desempenho 
Além disso, devem implementar scripts de teste em Python utilizando a ferramenta Pytest e apresentar o trabalho em sala de aula. 

 

Etapas do Trabalho 

1. Levantamento de Requisitos 

O que fazer: 
Conversar com o "cliente" para entender as necessidades do sistema. 
Definir as funcionalidades principais (ex: cadastro, autenticação, relatórios). 
Listar os requisitos funcionais (o que o sistema deve fazer) e não funcionais (desempenho, segurança, etc.). 
Entregável: 
Documento de requisitos. 
2. Elaboração do Plano de Testes 

O que fazer: 
Definir os objetivos dos testes (o que será testado e por quê). 
Crie os planos de testes a serem realizados (integração, sistema, aceitação, regressão, desempenho). 
Criar um cronograma com prazos para cada etapa. 
Entregável: 
Plano de testes. 
3. Criação dos Casos de Teste 

O que fazer: 
Para cada funcionalidade, criar cenários de teste (ex: cadastro válido, cadastro inválido). 
Detalhar os passos para executar cada teste e o resultado esperado. 
Entregável: 
Casos de teste documentados. 
4. Implementação dos Testes Automatizados 

O que fazer: 
Escrever scripts de teste em Python usando Pytest. 
Cobrir todas as funcionalidades principais do sistema. 
Executar os testes automatizados e registrar os resultados. 
Entregável: 
Scripts de teste em Python. 
Relatórios de execução dos testes. 
5. Execução dos Testes Manuais 

O que fazer: 
Testar funcionalidades que não podem ser automatizadas (ex: usabilidade, fluxos complexos). 
Documentar defeitos encontrados e relatar à equipe. 
Entregável: 
Relatório de defeitos. 
6. Documentação dos Resultados 

O que fazer: 
Consolidar os resultados dos testes automatizados e manuais. 
Analisar os resultados e identificar possíveis melhorias. 
Preparar um relatório final com todas as informações. 
Entregável: 
Relatório final. 
7. Apresentação do Trabalho 

O que fazer: 
Preparar uma apresentação de 15 minutos para a turma. 
Mostrar os principais pontos do trabalho: requisitos, casos de teste, resultados e conclusões. 
Demonstrar a execução de pelo menos um teste automatizado. 
Entregável: 
Slides da apresentação.
